Section 01

Présentation du CSS

CSS (Cascading Style Sheets — Feuilles de style en cascade) est le langage utilisé pour décrire la présentation visuelle d'un document HTML. Là où HTML structure le contenu, CSS s'occupe de son apparence : couleurs, polices, mises en page, animations…

Définition W3C
CSS est un mécanisme de feuilles de style permettant aux auteurs et aux utilisateurs de définir l'apparence des documents. CSS sépare la présentation du contenu, améliorant la maintenabilité, l'accessibilité et la performance des sites Web.

La cascade — pourquoi « Cascading » ?

Le terme « cascade » désigne l'algorithme par lequel le navigateur détermine quelle règle CSS s'applique à un élément lorsque plusieurs règles entrent en conflit. Trois critères sont évalués dans l'ordre :

  1. Origine & importance — Les styles !important des feuilles user-agent, utilisateur, puis auteur l'emportent sur les styles normaux dans l'ordre inverse.
  2. Spécificité — Plus un sélecteur est « précis », plus sa règle prime sur les autres.
  3. Ordre d'apparition — En cas d'égalité de spécificité, la dernière règle déclarée gagne.

Spécificité — la hiérarchie

La spécificité est calculée comme un score à 4 niveaux. Plus le score est élevé, plus la règle prime :

style="" Inline (1,0,0,0)
›
#id ID (0,1,0,0)
›
.class / :pseudo Classe (0,0,1,0)
›
element Type (0,0,0,1)
L'héritage : certaines propriétés CSS (color, font-family, font-size…) sont automatiquement héritées par les éléments enfants. D'autres (margin, padding, border…) ne le sont pas et doivent être redéclarées.
Section 02

Concepts fondamentaux

a) Sélecteur CSS

Un sélecteur CSS est la partie d'une règle CSS qui désigne le ou les éléments HTML auxquels s'appliquent les déclarations de style. Il « cible » des éléments selon leur type, leurs attributs, leur identifiant, leur classe, ou leur position dans l'arbre du document. 

/* Quelques sélecteurs fondamentaux */

p          { /* tous les paragraphes */ }
.carte     { /* tous les éléments ayant class="carte" */ }
#logo      { /* l'élément ayant id="logo" */ }
a:hover    { /* les liens au survol */ }
li::before { /* pseudo-élément avant chaque li */ }
CSS

b) Propriété CSS

Une propriété CSS est un mot-clé reconnu par le navigateur qui désigne un aspect visuel à modifier (couleur, taille, position, etc.). Il existe des centaines de propriétés. Voici les familles les plus courantes :

color

Couleur du texte

background-color

Couleur d'arrière-plan

font-family

Police de caractères

font-size

Taille de la police

font-weight

Graisse (bold, 700…)

margin

Marges extérieures

padding

Rembourrage intérieur

border

Bordure de l'élément

width / height

Largeur et hauteur

display

Mode d'affichage

position

Mode de positionnement

opacity

Transparence (0–1)

transition

Animation de changement

transform

Rotation, échelle, déplacement

box-shadow

Ombre portée

border-radius

Coins arrondis

c) Valeur CSS

Une valeur CSS est ce qui est assigné à une propriété. Les valeurs peuvent être des mots-clés, des nombres, des pourcentages, des unités, des couleurs, des fonctions… 

/* ── Mots-clés ── */
p { display: block; font-weight: bold; }

/* ── Unités de longueur ABSOLUES ── */
.titre { font-size: 24px; }   /* pixel — 1px = 1/96 inch */

/* ── Unités RELATIVES ── */
.texte { font-size: 1.2rem; }  /* relatif à l'élément racine (<html>) */
.enfant{ font-size: 1.2em;  }  /* relatif à la police de l'élément parent */
.hero  { width:   100%;   }   /* pourcentage du conteneur parent */
.plein { width:   100vw;  }   /* viewport width  */
.haut  { height:  100vh;  }   /* viewport height */

/* ── Couleurs ── */
h1 {
  color: red;                /* mot-clé */
  color: #e63946;            /* hexadécimal */
  color: rgb(230, 57, 70);   /* rgb() */
  color: rgba(230,57,70,.8); /* rgba() avec transparence */
  color: hsl(356,76%,56%);   /* teinte / saturation / luminosité */
}

/* ── Fonctions ── */
.fond {
  background: linear-gradient(135deg, #0d1117, #00d4aa);
  width: calc(100% - 40px);
  transform: rotate(45deg);
}

/* ── Variables CSS custom (CSS custom properties) ── */
:root {
  --couleur-primaire: #00d4aa;
  --taille-titre: 2.5rem;
  --espacement: 16px;
}
h1 {
  color:     var(--couleur-primaire);
  font-size: var(--taille-titre);
  margin:    var(--espacement);
}

d) Règle CSS

Une règle CSS est l'unité de base du CSS. Elle associe un sélecteur à un bloc de déclarations. Chaque déclaration est une paire propriété–valeur séparée par : et terminée par ;. 

/* ─────────────────────────────
   Anatomie d'une règle CSS
   ─────────────────────────────
   ┌── Sélecteur ──┐
   │               │
   ▼               │
.carte-projet ◄─────────────┘

{  ◄─── Accolade ouvrante (début du bloc de déclarations)

  ┌─────── Propriété
  │        ┌── Valeur        ┌─ Point-virgule obligatoire
  ▼        ▼                 ▼
  background-color: #161b22;
  color:            #e6edf3;
  padding:          24px;
  border-radius:    8px;
  border:           1px solid #30363d;

} ◄─── Accolade fermante
CSS
Il est possible de grouper plusieurs sélecteurs en les séparant par des virgules pour leur appliquer les mêmes déclarations : h1, h2, h3 { color: #00d4aa; }

e) Application du CSS

Il existe trois manières d'appliquer du CSS à un document HTML :

<!-- ──────────────────────────────────
  1. CSS EXTERNE (recommandé)
     Fichier .css séparé, lié via <link>
────────────────────────────────────── -->
<head>
  <link rel="stylesheet" href="style.css">
</head>

<!-- ──────────────────────────────────
  2. CSS INTERNE (dans le <head>)
     Utile pour les styles propres à UNE page
────────────────────────────────────── -->
<head>
  <style>
    body { font-family: sans-serif; }
    h1   { color: #00d4aa; }
  </style>
</head>

<!-- ──────────────────────────────────
  3. CSS INLINE (directement sur l'élément)
     À éviter — priorité maximale, non réutilisable
────────────────────────────────────── -->
<p style="color: red; font-size: 18px;">Texte rouge</p>
HTML
MéthodeEmplacementRéutilisableRecommandé
Externe (.css)Fichier séparé lié par <link>✅ Toutes les pages✅ Oui — bonne pratique
Interne (<style>)Dans le <head> de la page⚠️ Une seule page⚠️ Parfois (prototypage)
Inline (style="")Attribut de l'élément HTML❌ Un seul élément❌ À éviter

f) Sélecteur de type

Le sélecteur de type (ou sélecteur d'élément) cible tous les éléments d'un certain type HTML. C'est le sélecteur le moins spécifique. 

/* Cible TOUS les paragraphes de la page */
p {
  margin: 0 0 16px 0;
  line-height: 1.75;
}

/* Cible TOUS les liens */
a {
  color: #00d4aa;
  text-decoration: none;
}
a:hover {
  text-decoration: underline;
}

/* Cible TOUS les titres h1 à h3 */
h1, h2, h3 {
  font-family: 'Fraunces', serif;
  font-weight: 700;
  line-height: 1.2;
}

/* Sélecteur universel * (tous les éléments) */
* {
  box-sizing: border-box;
  margin: 0;
  padding: 0;
}
CSS

g) Sélecteur d'attribut

Les sélecteurs d'attribut ciblent des éléments en fonction de la présence ou de la valeur de leurs attributs HTML. Très puissants pour styliser des formulaires. 

/* Élément ayant cet attribut (quelle que soit sa valeur) */
input[required] {
  border-left: 3px solid #ff7b72;
}

/* Attribut ÉGAL à une valeur exacte */
input[type="text"]   { border-radius: 6px; }
input[type="email"]  { background-image: url('icons/email.svg'); }
input[type="submit"] { cursor: pointer; background: #00d4aa; }

/* Attribut CONTENANT la valeur (sous-chaîne) */
a[href*="github"] { color: #f0883e; }

/* Attribut COMMENÇANT par la valeur */
a[href^="https"] {
  padding-left: 18px;
  background: url('lock.svg') no-repeat left center;
}

/* Attribut FINISSANT par la valeur */
a[href$=".pdf"] {
  color: #ff7b72;
}

/* Attribut lang en français ou sous-langue fr-* */
[lang|="fr"] { font-style: italic; }
CSS

h) Sélecteur d'identificateur

Le sélecteur d'ID cible un élément unique identifié par son attribut id. Il est précédé du symbole # et a une spécificité très élevée. 

/* En HTML :  <header id="entete-principal"> */
#entete-principal {
  background:     #161b22;
  padding:        20px 40px;
  display:        flex;
  justify-content: space-between;
  align-items:    center;
}

/* En HTML :  <nav id="nav-principale"> */
#nav-principale {
  position: sticky;
  top:      0;
  z-index:  100;
}
CSS
Un id doit être unique dans une page HTML. Pour les styles réutilisables sur plusieurs éléments, préférez les classes. Évitez de surcharger les sélecteurs ID dans les grandes applications car leur haute spécificité rend les surcharges difficiles.

i) Sélecteur de classe

Le sélecteur de classe est le plus utilisé en CSS. Il cible tous les éléments ayant une certaine valeur dans leur attribut class. Il est précédé du symbole .. Un élément peut avoir plusieurs classes. 

/* En HTML :  <article class="carte-projet"> */
.carte-projet {
  background:    #161b22;
  border:        1px solid #30363d;
  border-radius: 10px;
  padding:       24px;
  transition:    transform .2s, box-shadow .2s;
}
.carte-projet:hover {
  transform:  translateY(-4px);
  box-shadow: 0 12px 30px rgba(0,0,0,.4);
}

/* Plusieurs classes sur un même élément :
   <button class="btn btn-primaire"> */
.btn {
  display:       inline-block;
  padding:       10px 24px;
  border:        none;
  border-radius: 6px;
  cursor:        pointer;
  font-weight:   600;
}
.btn-primaire { background: #00d4aa; color: #0d1117; }
.btn-danger   { background: #ff7b72; color: white; }
.btn-outline  { background: transparent; border: 2px solid #00d4aa; color: #00d4aa; }
CSS

j) Combinateurs

Les combinateurs permettent de créer des sélecteurs complexes en exprimant des relations entre éléments dans l'arbre DOM.

A B

Descendant — Cible B placé n'importe où à l'intérieur de A (enfant, petit-enfant…)

A > B

Enfant direct — Cible B qui est un enfant immédiat de A uniquement.

A + B

Frère adjacent — Cible B qui suit immédiatement A au même niveau.

A ~ B

Frères généraux — Cible tous les B frères qui suivent A.

/* Descendant : tous les li dans nav */
nav li {
  list-style: none;
  display:    inline-block;
}

/* Enfant direct : li enfants directs de ul seulement */
ul > li { padding: 8px; }

/* Frère adjacent : label qui suit immédiatement un input */
input:focus + label {
  color: #00d4aa;
  transform: translateY(-20px) scale(0.85);
}

/* Pseudo-classes structurelles */
li:first-child  { border-top: none; }
li:last-child   { border-bottom: none; }
li:nth-child(2n) { background: rgba(255,255,255,.04); } /* lignes paires */
a:not(.btn)     { text-decoration: underline; }
CSS

k) Modèle de boîte

Le modèle de boîte (Box Model) est le fondement du positionnement CSS. Chaque élément HTML est rendu comme une boîte rectangulaire composée de quatre zones concentriques :

MARGIN
BORDER
PADDING
CONTENT
width × height
.carte {
  /* ── CONTENT ── */
  width:  320px;   /* largeur du contenu */
  height: 200px;  /* hauteur du contenu */

  /* ── PADDING (rembourrage intérieur) ── */
  padding:        24px;              /* tous les côtés */
  padding:        16px 24px;        /* vertical | horizontal */
  padding:        10px 20px 14px 20px; /* haut | droite | bas | gauche */
  padding-top:    12px;
  padding-bottom: 12px;

  /* ── BORDER ── */
  border:        2px solid #30363d; /* épaisseur | style | couleur */
  border-top:    3px solid #00d4aa;
  border-radius: 8px;               /* coins arrondis */

  /* ── MARGIN (espace extérieur) ── */
  margin:        24px;
  margin:        0 auto;             /* centrage horizontal */
  margin-bottom: 32px;

  /* ── BOX-SIZING (très important !) ── */
  box-sizing: border-box; /* padding et border inclus dans width/height */
}
CSS
box-sizing: border-box est la propriété la plus importante du Box Model. Avec content-box (défaut), width ne compte que le contenu — padding et border s'y ajoutent. Avec border-box, width inclut tout : une boîte width: 300px fait toujours 300px, peu importe son padding ou border. Utilisez toujours * { box-sizing: border-box; }.

l) Boîtes flexibles (Flexbox)

Flexbox (Flexible Box Layout) est un système de mise en page CSS conçu pour disposer des éléments en une dimension (une ligne ou une colonne). Il résout élégamment des problèmes classiques : centrage vertical, distribution d'espace, réarrangement selon la taille d'écran.

Deux acteurs dans Flexbox
Le conteneur flex (flex container) — l'élément parent avec display: flex — contrôle la disposition de ses enfants.
Les éléments flex (flex items) — les enfants directs du conteneur — sont positionnés selon les règles du conteneur. 
/* ══ PROPRIÉTÉS DU CONTENEUR FLEX ══ */

.conteneur {
  display:         flex;          /* Active Flexbox */

  /* Direction */
  flex-direction:  row;           /* → gauche à droite (défaut) */
  flex-direction:  row-reverse;   /* ← droite à gauche */
  flex-direction:  column;        /* ↓ haut en bas */
  flex-direction:  column-reverse;/* ↑ bas en haut */

  /* Retour à la ligne */
  flex-wrap:       nowrap;        /* sur 1 ligne (défaut) */
  flex-wrap:       wrap;          /* passage à la ligne */

  /* Alignement sur l'AXE PRINCIPAL */
  justify-content: flex-start;   /* début */
  justify-content: flex-end;     /* fin */
  justify-content: center;       /* centré */
  justify-content: space-between;/* espacement égal entre éléments */
  justify-content: space-around; /* espacement autour des éléments */
  justify-content: space-evenly; /* espacement parfaitement égal */

  /* Alignement sur l'AXE TRANSVERSAL */
  align-items:     stretch;      /* s'étire pour remplir (défaut) */
  align-items:     flex-start;   /* aligné en haut */
  align-items:     flex-end;     /* aligné en bas */
  align-items:     center;       /* centré verticalement */

  /* Espace entre les éléments flex */
  gap:             16px;
  gap:             16px 24px;    /* row-gap | column-gap */
}

/* ══ PROPRIÉTÉS DES ÉLÉMENTS FLEX ══ */

.item {
  /* Croissance — occupe l'espace disponible */
  flex-grow:   1;    /* tous les items grandissent également */
  flex-grow:   2;    /* prend 2x plus de place que flex-grow: 1 */

  /* Rétrécissement */
  flex-shrink: 0;    /* ne rétrécit jamais */

  /* Taille de base */
  flex-basis:  200px;

  /* Raccourci : grow | shrink | basis */
  flex:        1 1 auto;
  flex:        1;               /* équivalent à 1 1 0 */

  /* Alignement individuel (surcharge align-items) */
  align-self:  center;
}
CSS
