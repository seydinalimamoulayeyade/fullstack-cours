Section 01

Présentation de JavaScript

JavaScript (JS) est le seul langage de programmation nativement exécuté par tous les navigateurs Web. Créé en 1995 par Brendan Eich chez Netscape en seulement 10 jours, il est aujourd'hui le langage le plus utilisé au monde et s'étend bien au-delà du navigateur grâce à Node.js.

Définition — ECMAScript
JavaScript est un langage de script interprété, dynamiquement typé, à prototype, monothread et orienté événements. Sa spécification officielle est ECMAScript, maintenue par le comité TC39 de l'ECMA International. ES6 (2015) représente la modernisation majeure du langage.

Section 02

Inclusion du JavaScript

Il existe trois façons d'intégrer du JavaScript dans une page HTML. Chacune a un usage précis.

01 Script externe (recommandé)

Le script est dans un fichier .js séparé, lié via la balise <script src>. C'est la méthode professionnelle : séparation des responsabilités, cache navigateur, maintenabilité.

<!-- Placé avant </body> : le DOM est entièrement chargé -->
<body>
  <!-- Contenu HTML -->

  <script src="js/portfolio.js"></script>
</body>

<!-- OU dans <head> avec l'attribut defer -->
<head>
  <!-- defer : télécharge en parallèle, exécute après parsing HTML -->
  <script src="js/portfolio.js" defer></script>

  <!-- async : télécharge en parallèle, exécute dès que prêt (ordre non garanti) -->
  <script src="js/analytics.js" async></script>
</head>
HTML
AttributTéléchargementExécutionUsage

asyncParallèleDès que prêt (sans ordre)Analytics, widgets tiers

02 Script interne

<body>
  <!-- Contenu HTML -->
  <script>
    // Code JS directement dans la page HTML
    // Utile pour un prototypage rapide ou un script court

e Fonctions (arrow, callback…)

En JavaScript, les fonctions sont des citoyens de première classe : elles peuvent être stockées dans des variables, passées en argument et retournées par d'autres fonctions.

// ── 1. Déclaration classique ──
// Hissée (hoisting) : peut être appelée AVANT sa définition
function direBonjour(prenom) {
  return `Bonjour, ${prenom} !`;
}
direBonjour('Moussa');  // "Bonjour, Moussa !"

// ── 2. Expression de fonction ──
// Non hissée : doit être définie avant l'appel
const additionner = function(a, b) {
  return a + b;
};

// ── 3. Arrow function / Fléchée (ES6) ──
// Syntaxe concise, hérite du this parent
const multiplier  = (a, b) => a * b;         // retour implicite
const carre       = n       => n ** 2;        // 1 paramètre → pas de ()
const coucou      = ()      => 'Coucou !';    // 0 paramètre

const creerProjet = (id, libelle, image) => {  // corps avec {}
  return { id, libelle, image, date: new Date().toISOString() };
};

// ── 4. Paramètres par défaut ──
function creerBouton(texte = 'Cliquer', type = 'button') {
  return `<button type="${type}">${texte}</button>`;
}
creerBouton();                 // type="button">Cliquer
creerBouton('Envoyer', 'submit');

// ── 5. Callback ──
// Une fonction passée en argument à une autre fonction
function executerAvecDelai(callback, delai) {
  setTimeout(callback, delai);
}

executerAvecDelai(() => {
  console.log('Exécuté après 2 secondes');
}, 2000);

// Callback dans les méthodes de tableau (section h)
projets.forEach(p => afficherCarte(p));  // forEach attend un callback
projets.filter(p => p.annee === 2025);    // filtre avec callback

// ── 6. Fonction auto-invoquée (IIFE) ──
// Exécutée immédiatement, crée une portée isolée
(function() {
  console.log('Initialisation du portfolio...');
  chargerProjets();
})();

// ── 7. Fermeture (Closure) ──
function creerCompteur() {
  let total = 0;                         // variable privée
  return () => ++total;                 // closure : accès à total
}
const compterProjets = creerCompteur();
compterProjets(); // 1
compterProjets(); // 2

g Objets

Un objet est une collection de paires clé–valeur (propriétés). C'est la structure de données centrale de JavaScript. Tout (tableau, fonction, DOM) est un objet.

// ── Création littérale ──
const projet = {
  id:           1,
  libelle:      'Portfolio Web',
  description:  'Application de gestion de projets',
  technologies: ['HTML', 'Tailwind', 'JavaScript'],
  image:        'images/portfolio.jpg',
  actif:        true,
  auteur:       { nom: 'Moussa', email: 'm@ex.sn' },

  // Méthode (fonction dans un objet)
  afficher() {
    return `[${this.id}] ${this.libelle}`;
  },
  estRecent() {
    return new Date(this.dateCreation).getFullYear() === new Date().getFullYear();
  }
};

// ── Accès aux propriétés ──
projet.libelle;           // notation pointée
projet['libelle'];        // notation crochets (utile avec des clés dynamiques)
projet.auteur.email;      // accès imbriqué
projet.afficher();        // appel de méthode → '[1] Portfolio Web'

// ── Modifier / Ajouter / Supprimer ──
projet.libelle    = 'Nouveau libellé';  // modifier
projet.annee      = 2025;              // ajouter
delete projet.actif;                    // supprimer

// ── Destructuring (ES6) — extraire des propriétés ──
const { id, libelle, technologies: techs } = projet;
console.log(id);      // 1
console.log(libelle); // 'Portfolio Web'
console.log(techs);   // ['HTML', 'Tailwind', ...]

// Avec valeur par défaut
const { note = 0, libelle: titre } = projet;

// ── Spread operator — copier / fusionner ──
const copie          = { ...projet };
const projetModifie  = { ...projet, libelle: 'Modifié', version: 2 };

// ── Méthodes utiles ──
Object.keys(projet);     // ['id', 'libelle', 'description', ...]
Object.values(projet);   // [1, 'Portfolio Web', ...]
Object.entries(projet);  // [['id',1], ['libelle','Portfolio Web'], ...]
Object.assign({}, projet, { actif: false }); // fusion (mutation du 1er)

// ── Raccourci de propriété (ES6) ──
const id2 = 5;
const lib = 'Blog';
const obj = { id2, lib };  // { id2: 5, lib: 'Blog' }

h Objet Array

Array est le type tableau de JavaScript. Les tableaux sont des objets dont les clés sont des indices numériques. Ils possèdent de nombreuses méthodes pour parcourir, transformer et rechercher des données. 

// ── Créer un tableau ──

// ── Accès ──
projets[0];                // premier élément
projets.at(-1);             // dernier (ES2022)
projets.length;             // nombre d'éléments

// ── Mutation (modifient le tableau original) ──
projets.push({ id: 1, libelle: 'Blog' });    // ajoute à la fin → retourne length
projets.pop();                                  // supprime le dernier → retourne élément
projets.unshift({ id: 0, libelle: 'Intro' });  // ajoute au début
projets.shift();                                 // supprime le premier
projets.splice(1, 1);                            // supprime 1 élément à l'index 1
projets.splice(2, 0, nouveauProjet);           // insère sans supprimer
projets.sort((a, b) => a.id - b.id);          // trier
projets.reverse();                               // inverser

// ── Méthodes fonctionnelles (retournent un NOUVEAU tableau) ──

// forEach — itérer sans retour
projets.forEach((p, index) => {
  afficherCarte(p, index);

// map — transformer chaque élément
const libelles = projets.map(p => p.libelle);

// findIndex — index du premier correspondant

// reduce — accumuler en une valeur
const nbTechs = projets.reduce((acc, p) => acc + p.technologies.length, 0);

// some / every
projets.some(p  => p.actif);    // au moins un actif ?
projets.every(p => p.id > 0);  // tous ont un id positif ?

// includes
techs.includes('React');        // true / false

// flat / flatMap
const toutesLestech = projets.flatMap(p => p.technologies);
// ['HTML','CSS','JS','React','Node',...]

// Spread + filter pour "supprimer" proprement
projets = projets.filter(p => p.id !== idASupprimer);

// Destructuring de tableau
const [premier, deuxieme, ...reste] = projets;

async function getTousProjets() {
  try {
    const response = await fetch('http://localhost:3000/projets');
    if (!response.ok) throw new Error(`HTTP ${response.status}`);

/** Afficher une notification toast temporaire */
function notifier(message, type = 'success') {
  const couleurs = {
    success: 'bg-green-600',
    error:   'bg-red-600',
    info:    'bg-blue-600',

  const toast = document.createElement('div');

     text-sm font-semibold animate-pulse`;
  toast.textContent = message;
  zoneNotif.appendChild(toast);
  setTimeout(() => toast.remove(), 3500);
}

/* ══════════════════════════════════════════════════
   ⑥ ajouterProjet(donnees)
      Ajoute un projet en mémoire + DOM + serveur
   ══════════════════════════════════════════════════ */
async function ajouterProjet(donnees) {
  try {
    // 1. Persister côté serveur (json-server génère l'id)
    const projetCree = await fetchPostProjet(donnees);

    // 2. Ajouter en mémoire (tableau projets)
    projets.push(projetCree);

    // 3. Ajouter sur l'interface (insérer le nœud en 1er)
    creerProjet(projetCree.id, projetCree.libelle, projetCree.image);

    // 4. Feedback utilisateur
    notifier(`✅ "${${projetCree.libelle}}" ajouté avec succès !`);

    // 5. Retourner à la liste
    formProjet.reset();
    afficherVue(vueListe);

  } catch (err) {
    console.error('ajouterProjet :', err);
    notifier('❌ Erreur lors de l\'ajout. json-server est-il démarré ?', 'error');
  }
}

          <h3 class="text-lg font-semibold text-yellow-400 mb-2">Description</h3>
          <p class="text-slate-400 leading-relaxed">
            ${projet.description || 'Aucune description.'}
          </p>
        </div>
        <div>
          <h3 class="text-lg font-semibold text-yellow-400 mb-3">
            Technologies utilisées
          </h3>
          <ol class="space-y-2">${listeTechs || '<li class="text-slate-500">Aucune</li>'}</ol>
        </div>
        <div class="flex gap-4 pt-4 border-t border-slate-800">
          <button onclick="afficherVue(vueListe)"
                  class="px-6 py-2 border border-slate-600 text-slate-300
                         rounded-lg hover:border-slate-400 transition-colors">
            ← Retour à la liste
          </button>
          <button onclick="supprimerProjet(${projet.id})"
                  class="px-6 py-2 border border-red-500/50 text-red-400
                         rounded-lg hover:bg-red-500 hover:text-white transition-all">
            Supprimer ce projet
          </button>
        </div>
      </div>
    </div>
  `;

  // Afficher la vue détail
  afficherVue(vueDetail);
}

/* ══════════════════════════════════════════════════
   ⑧ supprimerProjet(id)
      Supprime un projet de la mémoire, de l'interface et du serveur
   ══════════════════════════════════════════════════ */
async function supprimerProjet(id) {
  const idNum   = Number(id);
  const projet  = projets.find(p => p.id === idNum);

  if (!projet) return;

  if (!confirm(`Supprimer "${projet.libelle}" définitivement ?`)) return;

  try {
    // 1. Supprimer côté serveur
    const ok = await fetchDeleteProjet(idNum);
    if (!ok) throw new Error('Suppression serveur échouée');

    // 2. Supprimer du tableau mémoire
    projets = projets.filter(p => p.id !== idNum);

    // 3. Supprimer de l'interface
    const carteDOM = conteneurProjets.querySelector(`[data-id="${idNum}"]`);
    carteDOM?.remove();

    // 4. Si on était sur la vue détail → retourner à la liste
    if (!vueDetail.classList.contains('hidden')) {
      afficherVue(vueListe);
    }

    // 5. Vérifier si la liste est vide
    if (projets.length === 0) rendreListeProjets([]);

    notifier(`🗑️ "${projet.libelle}" supprimé.`);

  } catch (err) {
    console.error('supprimerProjet :', err);
    notifier('❌ Erreur lors de la suppression.', 'error');
  }
}

formProjet.addEventListener('submit', async (e) => {
  e.preventDefault();

  // Récupérer les valeurs
  const libelle      = inpLibelle.value.trim();
  const image        = inpImage.value.trim();
  const description  = inpDescription.value.trim();
  const technologies = [...formProjet.querySelectorAll('[name="tech"]:checked')]
                          .map(el => el.value);

  // Réinitialiser les erreurs
  setErreur('err-libelle', '');
  setErreur('err-image', '');

  // Validation
  let valide = true;
  if (libelle.length < 3) {
    setErreur('err-libelle', 'Le libellé doit comporter au moins 3 caractères.');
    valide = false;
  }
  if (!image) {
    setErreur('err-image', 'La source de l\'image est requise.');
    valide = false;
  }
  if (!valide) return;

  await ajouterProjet({ libelle, image, description, technologies });
});
